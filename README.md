# algo-dapp-template
A template for developers who want to build dapps on Algorand leading blockchain

- Connect to wallet
- Call the application
- Read from the application

## Installation:
```
npm i
```

## Build:
```
npm run build
```

## Testing
```
npm run test
```

Then open the http://localhost:8000 in your local browser
